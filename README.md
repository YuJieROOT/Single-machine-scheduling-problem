# Flask开发Web应用
## 项目结构
- **/project**: 项目根目录。
  - **/static**: 静态文件夹，存放CSS、JavaScript、图片等文件。
    - **/css**: 存放样式表文件。
    - **/js**: 存放JavaScript文件。
    - **/images**: 存放图片文件。
  - **/templates**: 模板文件夹，存放HTML模板文件。
    - **index.html**: 首页模板。
    - **about.html**: 关于页面模板。
  - **app.py**: Flask应用的主文件。
